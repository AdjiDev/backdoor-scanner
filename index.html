Getting Started

Installation

Install the package using npm:

npm install @adjidev/backdoor-scanner

Basic Usage

Here's how to perform a basic scan:

const { performScan } = require('@adjidev/backdoor-scanner');

// Scan files
const results = performScan(['file1.js', 'file2.php'], {
    deepScan: true,
    scanNodeModules: false,
    maxFileSizeMB: 5
});

console.log(results);

Scanning Content Directly

You can also scan code content directly:

const { scanContent } = require('@adjidev/backdoor-scanner');

const maliciousCode = "eval('malicious payload');";
const threats = scanContent(maliciousCode, { deepScan: true });

console.log(threats);

API Reference

performScan(files, options)

Scans multiple files for backdoors and malicious patterns.

Parameters

Name Type Description
files Array<string> Array of file paths to scan
options Object
  • deepScan: Boolean (default: true)
  • scanNodeModules: Boolean (default: false)
  • maxFileSizeMB: Number (default: 5)

Returns

Array of scan result objects containing:

  • file: Path to scanned file
  • threats: Array of detected threats
  • severity: Severity level (critical, high, medium)
  • threatScore: Numerical threat score (0-100)

scanContent(content, options)

Scans a string of code content for malicious patterns.

Parameters

Name Type Description
content string Code content to scan
options Object
  • deepScan: Boolean (default: true)

Tutorial

1. Setting Up the Scanner

First, install the package and require it in your project:

const { performScan, scanContent } = require('@adjidev/backdoor-scanner');

2. Scanning a Project Directory

Here's how to scan all JavaScript and PHP files in a directory:

const fs = require('fs');
const path = require('path');

function scanDirectory(dir) {
    const files = fs.readdirSync(dir)
        .filter(file => /\.(js|php)$/i.test(file))
        .map(file => path.join(dir, file));
    
    return performScan(files, { deepScan: true });
}

const results = scanDirectory('./src');
console.log('Found threats:', results.filter(r => r.threats.length > 0).length);

3. Integrating with CI/CD

You can integrate the scanner into your CI pipeline to fail builds when threats are detected:

const { performScan } = require('@adjidev/backdoor-scanner');
const filesToScan = ['src/**/*.js', 'src/**/*.php'];

const results = performScan(filesToScan);
const criticalThreats = results.filter(r => r.severity === 'critical');

if (criticalThreats.length > 0) {
    console.error('Critical threats detected! Build failed.');
    process.exit(1);
} else {
    console.log('No critical threats detected. Build passed.');
}

Frequently Asked Questions

What types of threats can the scanner detect?

The scanner can detect various threats including:

  • Obfuscated malicious code
  • Common backdoor patterns
  • Suspicious function calls (eval, system, etc.)
  • Malicious URLs and domains
  • XOR encoded payloads

Is this scanner suitable for production use?

This is currently a demo version. While it includes comprehensive detection patterns, it's recommended to:

  • Review all findings manually
  • Test thoroughly in your environment
  • Combine with other security tools
